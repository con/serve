---
title: "JSON-LD"
description: "JSON for Linked Data -- semantic metadata format used by datalad-metalad and RO-Crate"
---

[JSON-LD](https://json-ld.org/) extends JSON with linked data semantics
(context, types, identifiers).
Used by datalad-metalad for extracted metadata,
RO-Crate for research object descriptions,
and Neurobagel for semantic graph databases.
